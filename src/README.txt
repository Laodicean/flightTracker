====================================
	  Instructions for Set Up
====================================

This repository contains the code the Flight Tracker application as specified at 
www.cse.unsw.edu.au/~se2011. This is the final version of Group 6's assignment.

In order to run the GUI for this code follow the following steps:

1. In a terminal window run the file server.py using Python 3. Give as command line arguement the file name of the file containing the data for your flights. The flight data file must be a text file containing flights in the format specific in the specification. If no arguement is given the flights used will be a default set in testfiles.

2. Running server.py will host the GUI at localhost:8080.

3. Using your browser, go to localhost:8080

4. Use the GUI according to the instructions displayed on screen.
